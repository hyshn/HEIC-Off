# HEIC-Off

A lightweight Windows tool to batch convert HEIC images to JPG quickly and easily.

## ✨ Features

- 📁 Batch convert all HEIC files in a folder
- 🗂️ Preserves original folder structure
- 📋 Real-time progress log and status display
- ⏸️ Supports Pause / Stop during conversion
- 🔄 Option to also copy non-HEIC images (JPG, PNG, etc.)
- 🔌 Works fully offline — no internet required
- 🪟 Clean and simple interface

## 📥 Download

Go to [Releases](../../releases/latest) to download the latest `.exe`.  
No installation required — just run directly.

## 🚀 How to Use

1. Click **Browse** to select the **Source Folder** (folder containing your HEIC photos)
2. Click **Browse** to select the **Output Location** (where converted files will be saved)
3. (Optional) Check **"Also copy non-HEIC images"** to include JPG/PNG files as well
4. Click **Start Conversion**
5. Monitor progress in real time — when done, click **Open Result Folder**

> Output folder is automatically named `[SourceFolder]_jpg` and saved to your chosen location.

## 💻 System Requirements

- Windows 10 / 11
- No additional dependencies needed
